Могу я вместо mocks вставить какие-то данные, чтобы они отобразились в таблице?


Да, конечно. Вы можете вставить любые значения, которые соответствуют структуре данных, 
ожидаемой вашим приложением. Поскольку вы используете console.table(), 
он отображает JavaScript-объекты и массивы в виде таблицы — 
подойдут обычные строки, числа, булевы значения.

Какой тип данных здесь подойдёт?

Подойдут примитивные типы:

string — для Браузер, Разрешение, Проблема, Описание
number — для Номер
при желании: boolean или null, если нужно протестировать разные случаи

Подойдёт ли файл .csv?

Нет, напрямую — нельзя. В JavaScript-коде вы не можете просто 
"вставить .csv файл" вместо mocks. Но вы можете загрузить данные из .csv 
файла асинхронно (например, через Node.js или браузер с сервером), 
распарсить его и использовать для заполнения data. 
Однако в текущем контексте (простой скрипт в консоли) лучше использовать встроенные данные.

Пример: замена mocks на осмысленные данные
JavaScript
function Tmp(Номер, Браузер, Разрешение, Проблема, Описание) {
  this.N = Номер;
  this.Браузер = Браузер;
  this.Разрешение = Разрешение;
  this.Проблема = Проблема;
  this.Описание = Описание;
}

const INT = 8;
const data = [];

const browsers = ['Chrome', 'Firefox', 'Safari', 'Edge'];
const resolutions = ['1920x1080', '1366x768', '360x640', '1536x864'];
const issues = ['Отображение', 'Производительность', 'Анимация', 'Верстка сломана'];

for (let i = 1; i < INT; i++) {
  data.push(
    new Tmp(
      i,
      browsers[i % browsers.length],
      resolutions[i % resolutions.length],
      issues[i % issues.length],
      `Тестовый кейс для сценария #${i}`
    )
  );
}

console.table(data);
Этот код выведет таблицу в консоль с разнообразными данными.

const fs = require('fs');

const results = [];
